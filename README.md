# aida-core/.github

This repo holds **community health files** and the **org profile** for the [AIDA](https://github.com/aida-core) organization.

GitHub uses files in this repo as defaults for every other repo in the org that doesn't define its own copy. Touch a file here and you change the default for the whole org — please review changes carefully.

## What's in here

| Path | Purpose |
|---|---|
| [`profile/README.md`](profile/README.md) | Renders on the public org page at <https://github.com/aida-core> |
| [`CODE_OF_CONDUCT.md`](CODE_OF_CONDUCT.md) | Org-wide code of conduct |
| [`CONTRIBUTING.md`](CONTRIBUTING.md) | Default contribution guide |
| [`SECURITY.md`](SECURITY.md) | Vulnerability disclosure policy |
| [`SUPPORT.md`](SUPPORT.md) | Where to get help |
| [`ISSUE_TEMPLATE/`](ISSUE_TEMPLATE/) | Default issue templates |
| [`PULL_REQUEST_TEMPLATE.md`](PULL_REQUEST_TEMPLATE.md) | Default PR template |

## Editing

Changes here ship as soon as they merge to `main`. Use a feature branch and a PR even for small edits so the impact is visible to other maintainers.
